CSS PSEUDO-ELEMENTS..!

The CSS Pseudo-elements allow u to style the elements or parts of the elements without adding any IDs or classes to them

CCS-3 introduceda new double-colon(::) syntax for pseudo-elements to distinguish between them and pseudo-classes

selector::pseudo-element {property:value;}
